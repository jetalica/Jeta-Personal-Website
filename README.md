# Personal Website

## Development

Requirements: Node.js and npm

Steps:

```sh
# 1) Install dependencies
npm i

# 2) Run the dev server
npm run dev

# 3) Build for production
npm run build

# 4) Preview the production build
npm run preview
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
